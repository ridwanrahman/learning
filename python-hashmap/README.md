#### Question: why is accessing items in array/lists/tuples/sets (sequence types) so easy?

---
Sequence types in python are backed by an array which is a random-access data structure.
This follows two principles:
1. the array occupies a contiguous block of memory
2. every element in the array has a fixed size known upfront.
So, there is a very easy formula to calculate what is in that position.
